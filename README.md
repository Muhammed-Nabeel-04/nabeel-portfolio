# Muhammed Nabeel — Full Stack AI Developer Portfolio

A high-fidelity, interactive portfolio website featuring a "Neural Brain" thought-stream, a 3D tilting terminal, and reactive identity components.

## 🚀 Live Demo
[Check it out on GitHub Pages](https://nabeel-md.github.io/portfolio) *(Update with your actual URL)*

## ✨ Key Features
- **Neural Thought Stream**: A live terminal animation synchronized with a "Neural Brain" component that expresses developer reactions (Vibe Coding).
- **3D Tilt Interaction**: Smooth, mouse-reactive 3D perspective on the hero terminal and identity cards.
- **Reactive Cursor**: Custom cursor system with dynamic labeling and icon popups based on hovered elements.
- **AJAX Messaging**: Integrated contact form powered by Formspree with real-time terminal feedback (no page reloads).
- **Resume Hub**: Dedicated "View" and "Download" actions for quick resume access.
- **Modern Tech Stack Display**: Interactive skills grid covering AI, Data Science, Full-Stack Development, and DevOps.
- **Responsive Design**: Fully optimized for mobile and desktop viewing with a stabilized mobile navigation system.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Visuals**: CSS 3D Transforms, Canvas API (Aurora background)
- **Messaging**: [Formspree](https://formspree.io/) (ID: `xeozaweo`)
- **Deployment**: Optimized for GitHub Pages

## 📂 Structure
- `index.html`: Main structure and content.
- `style.css`: Modern glassmorphism UI, animations, and layouts.
- `script.js`: Terminal simulation logic, 3D tilt, custom cursor, and AJAX form handling.
