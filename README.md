## ✨ Key Features
- **Neural Thought Stream**: A live terminal animation synchronized with a "Neural Brain" component that expresses developer reactions (Vibe Coding).
- **3D Tilt Interaction**: Smooth, mouse-reactive 3D perspective on the hero terminal and identity cards.
- **Reactive Cursor**: Custom cursor system with dynamic labeling and icon popups based on hovered elements.
- **Modern Tech Stack Display**: Interactive skills grid covering AI, Data Science, Full-Stack Development, and DevOps.
- **Responsive Design**: Fully optimized for mobile and desktop viewing.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Visuals**: CSS 3D Transforms, Canvas API (Aurora background)
- **Deployment**: Optimized for GitHub Pages

## 📂 Structure
- `index.html`: Main structure and content.
- `style.css`: Modern glassmorphism UI, animations, and layouts.
- `script.js`: Terminal simulation logic, 3D tilt, and custom cursor system.
